# gstack

Use the `/browse` skill from gstack for all web browsing. **Never use `mcp__claude-in-chrome__*` tools.**

## Troubleshooting

If gstack skills aren't working, rebuild the binary and register skills:

```bash
cd .claude/skills/gstack && ./setup
```
