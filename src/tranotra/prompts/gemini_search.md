# Gemini Search Prompt

Search for soft PVC product manufacturers in {country} matching query: {query}

## Requirements

- 100+ employees
- Export to EU or USA markets
- Return ALL companies found

## Output Fields

Return each company with these 16 fields:

1. Company Name (English)
2. City/Province
3. Year Established
4. Employees (approximate: <100 / 100-500 / 500-2000 / 2000+)
5. Estimated Annual Revenue
6. Main Products (specific, e.g., "PVC garden hoses, flexible tubing")
7. Export Markets (which countries they sell to)
8. Export to EU/USA/Japan? (Yes/No)
9. Raw Materials (e.g., PVC resin, plasticizers)
10. Best Plasticizer for them (DOTP/DOP/ATBC/TOTM/DOS/TBC)
11. Why that plasticizer (1 sentence)
12. Company Website
13. Contact Email
14. LinkedIn Company Page URL
15. Best job title to contact (Purchasing Manager / Procurement Director / etc)
16. Prospect Score (1-10)

## Format

Return data in STRUCTURED format. Use whatever format (CSV, JSON, Markdown table) gives the clearest results.
